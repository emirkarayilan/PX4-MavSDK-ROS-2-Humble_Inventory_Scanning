# PX4-MavSDK-ROS-2-Humble_Inventory_Scanning
This repo contains the item counting via(OpenCV) and autonomous motion planing(MAVSDK) algorithm developed using PX4, MavSDK and ROS 2 Humble.

You can reach the video of the simulation via this link:
https://drive.google.com/file/d/15CTWJWmI1cfcAAbpkkO855p2a5grWahr/view

## VIO Tutorial (GPS-Free Navigation)

A comprehensive tutorial for using Visual Inertial Odometry (VIO) with PX4 Iris model without GPS is now available!

**See [VIO_TUTORIAL.md](VIO_TUTORIAL.md) for the complete guide.**

### Quick Start Files:
- **VIO_TUTORIAL.md** - Complete step-by-step tutorial from installation to flight
- **vio_test_flight.py** - Test script to verify VIO setup
- **mission_vio.py** - VIO-enabled warehouse inventory scanning mission
- **simple_vio_mock.py** - Mock VIO system for testing and development

### Features:
- Indoor navigation without GPS
- Vision-based position estimation
- Complete PX4 parameter configuration
- Example missions and test flights
- Troubleshooting guide
